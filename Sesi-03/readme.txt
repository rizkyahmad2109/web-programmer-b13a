Cara mengintegrasikan ke CSS

internal : memuliskan css kedalam tag head dan tag style 
			<head>
				<style>
					//tempan menuliskan css
				</style>
			</head>

inline: menuliskan css langsung kedalam tag html yang ingin diberikan style css
		<p style="color: red;">Belajar CSS</p>

eksternal : menuliskan css ke file css yang telah dibuat (style.css) dengan cara diintegrasikan dulu didalam file html
			<head>
				<link rel="stylesheet" href="style.css" />
 			</head>
