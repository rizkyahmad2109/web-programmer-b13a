Menghitung luas dan keliling lingkaran

  • Tahapan dalam menghitung luas dan keliling lingkaran :

    1. Mengisi nilai jari-jari
    2. Mengisi variabel Pi (rasio keliling lingkaran dengan diameternya) dengan nilai konstanta Pi yaitu 3.14
    3. Menghitung luas lingkaran dengan menggunakan rumus (luas = Pi x jari-jari x jari-jari)
    4. Menghitung keliling lingkaran dengan menggunakan rumus (keliling = 2 x Pi x jari-jari)
    5. Tampilkan hasil perhitungan luas lingkaran
    6. Tampilkan hasil perhitungan keliling lingkaran

  • Pseudocode menghitung luas dan keliling lingkaran :

    1. READ and SAVE jari2, pi, luas, keliling
    2. COMPUTE luas and keliling
    3. SAVE previous computation luas and keliling
    4. SHOW the computation luas and keliling
